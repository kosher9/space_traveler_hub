## installation

To get a local copy up and running follow these simple example steps.

- clone the repository by running
``` git@github.com:kosher9/space_traveler_hub.git ```
- navigate to the folder
``` cd <DIRECTORY> ```
- Install packages
``` npm i ```
- Run the app
``` npm start ```
